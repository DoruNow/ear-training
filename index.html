How to use the app Open

Resolve mode

  1. Press Start. The app establishes the key and plays one hidden note.
  2. Find the note on the piano before revealing it.
  3. Press Reveal. The app shows the degree and plays the path back to the nearest Do.
  4. Play that resolution path yourself.

Phrase mode

  1. Press Start. The app plays a short hidden phrase.
  2. Sing the contour before touching the piano.
  3. Find the phrase in the source key.
  4. Press Reveal, then play the same solfege in the transpose key.

Rhythm module: ratio tapping

Use a metronome at 60 BPM. Each click is one second, so 30 clicks = 30 seconds. This makes the metronome both a timer and a rhythm anchor.

2:1

One hand taps with the click. The other hand fits one extra tap between clicks.

3:1

One hand taps with the click. The other hand fits three even taps per click interval.

4:1

One hand taps with the click. The other hand fits four even taps per click interval.

Do each ratio twice: first with the right hand faster, then with the left hand faster. The click-hand must stay calm and exact.

The next level is permutation: keep the inner grid alive, but remove one or more taps. A 4:1 grid can become tap tap rest tap, rest tap tap tap, or tap rest tap tap. The rhythm is created by taking notes away from a continuous subdivision.

How this trains the real skill Open

The app gives you a small sound target. Your piano gives the answer. That keeps the training close to the real skill.

Movable Do is used: Do means the home note of the current key. In C, Do is C. In G, Do is G. The same solfege shape becomes different piano keys.

Resolve mode establishes the key, plays one target note, and lets you resolve it back to Do. Lower degrees resolve downward; upper degrees resolve upward to high Do.

Phrase mode uses the same movable-Do system, but gives you a short melody to find on piano and transpose.

Optional tasks are organized as a small task deck in the code. Add more cards to taskDeck without changing the playback engine.

The generator avoids chaotic note soup. Early phrases mostly move by steps, start near home, and return to stable notes.

Sample mode expects Salamander v8 files in piano-samples/, next to this HTML file. Use a local server, for example python -m http.server 8000, so the browser can load and shape the audio properly.
